# 💇‍♀️ Beauty Salon Management System

✨ A modern web-based application designed to streamline beauty salon operations including appointment booking, service management, and customer handling.

---

## 📌 Overview

The **Beauty Salon Management System** helps salons efficiently manage their daily activities such as scheduling appointments, tracking services, and improving customer experience.

It provides a user-friendly interface for both customers and salon administrators.

---

## 🚀 Features

* 📅 Online Appointment Booking
* 👩‍💼 Customer Management
* 💇 Service & Pricing Management
* ⏰ Real-time Availability Check
* 🔐 Secure Login & Authentication
* 📊 Dashboard for Admin
